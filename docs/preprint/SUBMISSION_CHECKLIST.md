# arXiv Submission Checklist

## Before submitting
- [ ] Create arXiv account at https://arxiv.org/user/register
- [ ] Choose primary category: cs.DL (Digital Libraries and Repositories)
- [ ] Choose cross-list: q-bio.QM, physics.soc-ph, cs.IR
- [ ] Convert usdr_preprint.md to LaTeX or upload as PDF
- [ ] Add all author information
- [ ] Verify all DOI citations are correct
