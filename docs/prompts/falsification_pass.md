# Prompt: falsification / red-team pass

Use before promoting work to **finding** status (see [METHODOLOGY.md](../METHODOLOGY.md)).

---

**Proposed claim or finding:** `[one paragraph]`

**Supporting evidence (summary):** `[bullets + links to methods/data]`

**Instructions:**

1. List the **strongest** reasons this claim could be **wrong**, including boring base-rate explanations.
2. Identify **confounders**, **selection effects**, and **leakage** if any ML/statistics are involved.
3. Propose **alternative models** that fit the same observations.
4. Suggest **checks** (robustness, ablations, independent replication paths) ranked by feasibility.
5. Rate **overall resilience** of the claim: fragile / moderate / robust—with **caveats**.

**Output format:** “How I’d try to break this” as a numbered list, then “What would change my mind.”
