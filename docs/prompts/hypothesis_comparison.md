# Prompt: hypothesis comparison

Use with [METHODOLOGY.md](../METHODOLOGY.md) and [VISION_AND_SCOPE.md](../VISION_AND_SCOPE.md).

---

**Hypotheses:**

- H1: `[statement]`
- H2: `[competing statement]`

**Tasks:**

1. Restate each hypothesis in falsifiable form. List **observable implications** that differ between H1 and H2.
2. For each hypothesis, list **existing evidence** (with citations) and **missing evidence**.
3. Propose **decisive or incremental experiments** (conceptual or concrete) that would shift belief—with expected costs/risks.
4. State which hypothesis is **more parsimonious** and in what domain that parsimony applies.
5. Explicitly list **what you would need to admit you were wrong** for the currently preferred hypothesis.

**Constraints:** Label speculation. Do not upgrade speculative reasoning to “findings.”
