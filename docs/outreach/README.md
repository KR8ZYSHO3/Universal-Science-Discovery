# Outreach & Launch Materials

This folder contains the complete, ready-to-execute materials for the USDR public launch (v0.5.0, targeted mid-2026).

## Primary Deliverable

- **[USDR_LAUNCH_OUTREACH_PACKAGE.md](USDR_LAUNCH_OUTREACH_PACKAGE.md)** — Everything in one place:
  - Complete 30-day Launch Calendar (pre-launch soft activities through mid-July follow-up)
  - Ready-to-post Reddit r/OpenScience flagship post
  - LinkedIn long + short versions
  - X/Twitter main thread + 5 shorter supporting posts
  - Researcher email/DM pitch templates (1 generic + 3 personalized for Physics, Biology, CS/AI-for-Science)
  - Full Response Playbook with template replies for 10+ common questions and pushback scenarios
  - Graphics ideas, suggested timings, hashtags, and CTAs for every asset

## Supporting Launch Documents (Repo Root)

- [LAUNCH_PLAYBOOK.md](../../LAUNCH_PLAYBOOK.md) — Strategic goals, narrative guardrails, audience priorities, channel strategy, asset checklist, and high-level phases.
- [LAUNCH_STATS_2026-06.md](../../LAUNCH_STATS_2026-06.md) — Baseline numbers (Phase 0 seed), launch window targets, channel-specific goals, and measurement cadence. Update daily during the active window.

## Quick Start for Launch Day

1. Update current entry counts and any new contributors in LAUNCH_STATS_2026-06.md.
2. Copy the exact posts from the Outreach Package.
3. Use the stagger timing in the calendar (GitHub release first, then X, LinkedIn, Reddit, email).
4. Reply to every comment/DM using the Response Playbook templates.
5. Log engagement and conversions back into LAUNCH_STATS.
