# Licensing notes

The default license for this repository is the [MIT License](../LICENSE).

## When to add exceptions

Document exceptions here when:

- **Specific files** use a different license (for example, adapted third-party text or figures).
- **Outputs** (preprints, reports, datasets) should be **CC BY** or another public license distinct from MIT for clarity.
- **Models or weights** you redistribute carry their own license terms.

## Per-artifact metadata

For each licensable artifact, prefer a short header or sidecar note:

- SPDX license identifier where applicable
- Attribution for derived work
- Link to upstream terms of use

## Default until amended

If this file contains only this section, **MIT applies repository-wide** with no recorded exceptions.
