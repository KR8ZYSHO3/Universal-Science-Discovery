    VISION_AND_SCOPE GitHub · Blueprint GitHub

    METHODOLOGY GitHub · Read before proposing scientific claims

    Ethics & data · Code of Conduct · LEGAL

    CONTRIBUTING · Use GitHub issues and pull requests · Prefer small changes: unknowns, hypotheses, bridges, reproducible stubs (see CONTRIBUTING)
