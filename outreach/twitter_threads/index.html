    X.com Thread Templates¶

    Three pre-written thread templates for promoting USDR on X (formerly Twitter). Replace [bracketed] placeholders before posting. Keep each tweet ≤ 280 characters.

    Thread 1: "What science doesn't know yet"¶

    Tweet 1 (hook)

    Science has conquered so much. Here's what it still can't explain — a thread 🧵

    Tweet 2

    1/ Why is there more matter than antimatter in the universe? Everything we observe suggests they were created equally at the Big Bang. One of the deepest open problems in physics. #OpenScience

    Tweet 3

    2/ How does a single fertilised cell become 37 trillion specialised cells — all with identical DNA — in a precisely choreographed sequence? Developmental biology still has no complete answer.

    Tweet 4

    3/ What is dark energy? It makes up ~68% of the universe and is driving accelerating expansion. We have a name for it. We have almost no idea what it actually is.

    Tweet 5

    4/ How did the first self-replicating molecule arise from chemistry alone? The origin of life remains an open problem spanning chemistry, physics, and information theory.

    Tweet 6

    5/ Why do we sleep? Every animal with a nervous system does it. The evolutionary cost is enormous. Neuroscience still lacks a complete mechanistic explanation.

    Tweet 7

    6/ What's remarkable: many of these unknowns share the same underlying mathematics. Percolation theory. Renormalisation group. Self-organised criticality. Different fields. Same equations. Nobody is connecting the dots at scale.

    Tweet 8 (CTA)

    7/ We're building a version-controlled, open-source map of what humanity doesn't know yet. 603 open questions. 22 domains. 49 cross-domain bridges. It's called USDR — and it's on GitHub. 👉 https://github.com/KR8ZYSHO3/Universal-Science-Discovery Contribute your field's unknowns.

    Thread 2: "The same math, everywhere"¶

    Tweet 1 (hook)

    The equation that explains bird migration also explains superconductors and stock market crashes. Here's why 🧵

    Tweet 2

    1/ Percolation theory asks: at what threshold does a random network become connected? Originally developed for porous media — water flowing through rock. It turns out to describe a shocking range of phenomena.

    Tweet 3

    2/ Ecology: Bird flocks and fish schools show phase transitions at a critical density. Below threshold → fragmented groups. Above → collective motion. Same math. Different substrate.

    Tweet 4

    3/ Physics: High-temperature superconductors exhibit percolation-like connectivity of superconducting domains at the critical temperature Tc. The phase transition is mathematically identical.

    Tweet 5

    4/ Finance: Market crashes propagate like percolation on an asset-correlation network. When connectivity exceeds the critical threshold, a local shock becomes systemic. The 2008 crisis followed this pattern.

    Tweet 6

    5/ Neuroscience: Neural avalanches in cortical networks obey power laws consistent with a system poised near a percolation critical point. The brain may operate near criticality by design.

    Tweet 7

    6/ This is a bridge — a formal mathematical correspondence between fields that rarely talk to each other. Ecologists and condensed matter physicists are solving the same equation. Neither community fully knows it.

    Tweet 8 (CTA)

    7/ We're cataloguing these bridges — all 49 of them — in an open-source knowledge graph. Each bridge has a formal translation table: source concept → target concept → mathematical object. 👉 https://kr8zysho3.github.io/Universal-Science-Discovery/dashboard/ Help us find the next one.

    Thread 3: "We're mapping the unknowns of science"¶

    Tweet 1 (hook)

    What if there was a GitHub for open scientific questions? We built it. 🧵

    Tweet 2

    1/ The Universal Science Discovery Repository (USDR) is a version-controlled, open-source catalog of what science doesn't yet know. Not what we've discovered — what we haven't.

    Tweet 3

    2/ Every entry is a structured YAML file with: - A precise statement of the unknown - The domain and sub-field - Links to primary literature - Connection to related hypotheses and cross-domain bridges Machine-readable. Human-curated.

    Tweet 4

    3/ Right now the repository contains: 📊 603 structured entries across 22 scientific domains 🔗 49 cross-domain bridges with formal mathematical translations 🕸️ A knowledge graph of 603 nodes and 628 edges

    Tweet 5

    4/ We built an AI co-pilot that analyses the graph and proposes novel bridge candidates — domain pairs with high open-question density and no existing formal bridge. It surfaces connections that no individual researcher would spot.

    Tweet 6

    5/ Everything is peer-reviewed via GitHub pull requests. No single expert can validate 22 domains. The governance model is distributed: domain stewards own quality in their field, just like open-source maintainers own their package.

    Tweet 7

    6/ We're looking for: 🔬 Domain experts to validate and extend unknowns 🌉 Cross-domain thinkers to propose new bridges 🛠️ Engineers to improve tooling and the dashboard 📝 Writers to help translate findings for broader audiences

    Tweet 8 (CTA)

    7/ This is infrastructure for the next generation of science. Star the repo. Submit an unknown. Review a bridge. 👉 https://github.com/KR8ZYSHO3/Universal-Science-Discovery 🖥️ https://kr8zysho3.github.io/Universal-Science-Discovery/dashboard/
