# Hypotheses

Versioned hypothesis registry. Typical workflow:

| Subfolder | Meaning |
|-----------|---------|
| `active/` | Under test or discussion |
| `validated/` | Strong support (still provisional vs “truth”) |
| `archived/` | Superseded or refuted |

Files should conform to [schemas/hypothesis.yaml](../schemas/hypothesis.yaml). Use [templates/hypothesis-pr-template.md](../templates/hypothesis-pr-template.md) in PRs.
