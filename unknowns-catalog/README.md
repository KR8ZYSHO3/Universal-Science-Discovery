# Unknowns catalog

Global and cross-cutting **research gaps**. Prefer stable `u-...` ids (see [schemas/unknown.yaml](../schemas/unknown.yaml)).

Suggested layout as volume grows:

- `high-priority/` — impact-ranked gaps
- `systematic-gaps/` — from literature scans
- `unknown-unknowns/` — speculative frontiers (clearly labeled)

Use [templates/unknown-issue-template.md](../templates/unknown-issue-template.md) when triaging new gaps.
