# Bridge: Synthetic Biology ↔ Astrobiology

**Core claim:** Synthetic lichen-like autotrophic-heterotrophic consortia engineered for Earth-based biofabrication are the same architecture that astrobiology requires for self-sustaining off-world resource utilisation.

**Key insight:** Astrobiology/ISRU engineers specify what a self-sustaining biological system must do in space; synthetic biologists have the circuit-design tools to build it; ecologists have the stability theory to ensure it doesn't collapse. None of the three communities routinely works with the others.
